# TOKENS — SPACING

## Ceremonial Scale. Stage Presence.

Sections breathe. `py-32` (128px) is the minimum. Everything centered.

---

## Scale

| Usage | Value | Tailwind |
|---|---|---|
| Section vertical | 128px | `py-32` |
| Hero padding | 160px | `py-40` |
| Card padding | 32px | `p-8` |
| Grid gaps | 32px | `gap-8` |
| Heading margin | 64px | `mb-16` |

## Symmetry: Always Default to Center

Hero, headings, CTAs — all centered. Bilateral symmetry is architectural law.
Feature grids: even column counts (2, 3). Never odd asymmetric layouts.
