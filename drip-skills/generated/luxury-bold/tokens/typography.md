## Opening rule
The typography features a single, consistent typeface across all text, focusing on Helvetica Neue for its modern, clean, and readable properties.

## Hierarchy Rules
1. Headings use Helvetica Neue at 700 weight to denote importance.
2. Body text always maintains a 400 weight for readability.
3. All headings are uppercase with tighter tracking.
4. Avoid using decorative fonts at any size.

## Icon Typography (if using Lucide or similar)
- strokeWidth value: Not applicable
- Icons remain minimal and outlined.

## What Never To Do
1. Never introduce additional fonts.
2. Avoid italics; maintain a uniform upright style.
3. Do not vary font weights within a single text element.
4. No decorative text effects like shadows or embossed looks.
