## Base Unit
The base unit is 8px, facilitating balance between clarity and structure.

## Spacing Scale
| Token | Value | Tailwind | Usage |
|--|--|--|--|
| xs | 8px | space-xs | Minor element spacing |
| sm | 16px | space-sm | General padding/margin |
| md | 24px | space-md | Form fields and buttons |
| lg | 32px | space-lg | Section padding |
| xl | 64px | space-xl | Large gaps between elements |

## Layout Widths
| Token | Value | Usage |
|--|--|--|
| container | 1200px | Max width for centered layout |

## Section Spacing Rules
- Vertical padding for sections is 48px.
- Heading to content maintains 32px separation.
- Uniform 24px spacing between form fields or interactive elements.

## Special Spacing Rules
1. Ensure an even distribution of negative space.
2. Maintain consistent gutter spacing in grid layouts.
3. Element padding should allow for comfortable readability, promoting uncluttered designs.
