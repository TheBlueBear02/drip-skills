## Mobile Adaptations
- Navbar collapses into a hamburger menu on smaller screens.
- Typography scales down to ensure readability.
- Grid collapses into a single column below 768px.
- Padding and margins reduce by a factor of half for compact displays.

## Touch Target Rules
- All interactive elements must have a minimum height of 48px.
- Elements should respond to touch without applying hover effects.
