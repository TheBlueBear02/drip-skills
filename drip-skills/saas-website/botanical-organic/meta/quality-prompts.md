# Quality Test Prompts — Botanical Organic

Use these prompts to run **Law 1 (Extrapolation Test)** before publishing or after major changes.

---

## Prompt 1: Modal Dialog

> Build a gentle confirmation modal for unsubscribing from a wellness newsletter.

**Pass criteria:**
- Paper grain texture present on page
- Modal uses rounded-3xl and soft green-tinted shadow
- Playfair Display heading with one italic emphasis word
- 500ms ease-out transitions minimum

---

## Prompt 2: Plant Care Schedule

> Build a weekly plant care schedule grid showing days and care tasks.

**Pass criteria:**
- Warm alabaster `#F9F8F4` background — not white
- Staggered card offsets in grid
- Terracotta `#C27B66` on interactive hover states
- Lucide icons at strokeWidth 1.5

---

## Prompt 3: Newsletter Signup

> Build an inline newsletter signup with email input and subscribe button.

**Pass criteria:**
- Pill-shaped button (rounded-full)
- Arch or rounded-3xl input container
- No hard offset shadows — diffused green-tinted only
- Source Sans 3 body typography
