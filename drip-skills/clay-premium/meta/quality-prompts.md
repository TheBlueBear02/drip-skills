# Quality Test Prompts — Clay Premium

Use these prompts to run **Law 1 (Extrapolation Test)** before publishing or after major changes.

---

## Prompt 1: Modal Dialog

> Build a playful confirmation modal with squish animation on the confirm button.

**Pass criteria:**
- 4-layer shadow stack on modal panel
- rounded-[20px] minimum on all corners
- Confirm button has hover lift + active squish
- Lavender canvas `#F4F1FA` visible behind modal

---

## Prompt 2: Volume Slider

> Build a volume slider control with a draggable thumb and track.

**Pass criteria:**
- Track is recessed (inset shadow), thumb is convex (outward shadow)
- Nunito Black label typography
- Super-rounded track and thumb shapes
- 200ms squish/bounce on thumb press

---

## Prompt 3: Achievement Badges Grid

> Build a grid of 6 achievement badges with gradient icon containers.

**Pass criteria:**
- Each badge has 4-layer shadow and rounded-[20px]+
- Icon containers use vivid gradient fills
- Animated blobs visible in page background
- Candy-store saturated colors — not muted pastels
